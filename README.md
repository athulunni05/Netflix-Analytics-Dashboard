# 🎬 Netflix Analytics Dashboard (Power BI)

## 📊 Project Overview
This project presents an interactive Netflix Analytics Dashboard built using Power BI to derive insights from complex datasets and support data-driven decision-making. The dashboard analyzes user engagement, content performance, viewing behavior, and recommendation effectiveness.

---

## 🎯 Key Insights
• User engagement, demographics, and subscription behavior  
• Content performance by genre, rating, and release trends  
• Viewing behavior by device type, quality, and geography  
• Search patterns and recommendation effectiveness  

---

## 🛠 Tools & Techniques
• Power BI  
• Data Modeling & DAX  
• Advanced Data Visualization  
• Analytical & Business Storytelling
